# Access and License Control (Public Demo)

This repository contains a sanitized demo version of an internal license-control workflow.

- Main documentation: `project.md`
- Demo project: `demo_public/`
- Entry point after generation: `demo_public/output/home.html`

## Quick start

```bash
cd demo_public
python -m pip install -r requirements.txt
python src/generate_demo_dashboard.py
```
